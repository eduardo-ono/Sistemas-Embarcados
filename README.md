* [Arduino IDE](https://www.arduino.cc)
* [TinkerCAD](https://www.tinkercad.com)
* [SimulIDE](https://www.simulide.com)
* Placa Arduino Uno (opcional)
* Placa Raspberry Pi 3B, 3B+ ou 4
